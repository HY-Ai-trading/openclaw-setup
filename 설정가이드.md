# 오픈클로 자동매매 설정 가이드

## 구성 개요

```
WSL / Linux 서버
└── ~/openclaw-setup/
    ├── scan.py           ← 핵심: 전 종목 분석 + 매매 판단 + trade.py 직접 실행
    ├── trade.py          ← 키움 API 지정가 주문 실행
    ├── query.py          ← API 서버 조회 래퍼 (인증 자동)
    ├── check_time.py     ← 장중 여부 확인
    ├── dart.py           ← DART 악재 공시 조회
    ├── .env              ← 시크릿 키 (git 제외)
    └── openclaw-config/
        ├── SOUL.md       ← 에이전트 시스템 프롬프트
        └── jobs.json     ← cron 잡 설정
```

---

## 1. 의존성 설치

```bash
pip install httpx python-dotenv
```

---

## 2. .env 설정

`.env.example`을 복사해서 실제 값으로 채우기:

```bash
cp .env.example .env
nano .env
```

| 항목 | 설명 |
|------|------|
| `TRADING_SERVER_URL` | 키움 API 서버 주소 |
| `SIGNAL_SECRET_KEY` | API 서버 HMAC 인증 키 |
| `DISCORD_BOT_TOKEN` | Discord Bot 토큰 |
| `DISCORD_USER_ID` | Discord DM 수신자 사용자 ID |
| `DASHBOARD_PASSWORD` | 대시보드 로그인 비밀번호 |
| `SESSION_SECRET` | 세션 암호화 랜덤 문자열 |
| `DART_API_KEY` | DART 공시 API 키 |

## 3. OpenClaw SOUL 등록

```bash
cp openclaw-config/SOUL.md ~/.openclaw/workspace/SOUL.md
```

`SOUL.md` 안의 `YOUR_USERNAME`을 실제 리눅스 사용자명으로 치환:

```bash
sed -i 's|YOUR_USERNAME|'$USER'|g' ~/.openclaw/workspace/SOUL.md
```

---

## 4. Cron 잡 등록

```bash
cp openclaw-config/jobs.json ~/.openclaw/cron/jobs.json
```

`jobs.json`에서 아래 플레이스홀더를 실제 값으로 수정:

```json
"YOUR_USERNAME"        → 실제 리눅스 사용자명
"YOUR_DISCORD_USER_ID" → Discord 사용자 ID
"YOUR_JOB_UUID_1/2"   → python3 -c "import uuid; print(uuid.uuid4())"
```

등록된 잡:
- `trading-8h50`: 평일 08:50 — 장 시작 전 점검
- `trading-9h-to-15h20`: 평일 09:00~15:20 20분마다 — 장중 분석

## 5. 수동 테스트

**API 연결 확인:**
```bash
python3 query.py /kiwoom/account
python3 query.py /kiwoom/holdings
```

**전 종목 분석 실행:**
```bash
python3 -u scan.py
```

scan.py가 매매 신호를 판단하면 trade.py를 직접 subprocess로 실행해 주문까지 처리함.

**특정 종목 수동 주문:**
```bash
python3 trade.py --code 005930 --name 삼성전자 --action BUY \
  --confidence 0.75 --ratio 0.5 --reason "테스트"
```

## 7. 전체 흐름

```
평일 8:50 / 9:00~15:20 Cron 자동 실행
    ↓
check_time.py → 장외시간이면 SKIP 종료
    ↓
scan.py 실행
    ├── /kiwoom/account     예수금 + 보유현황 조회
    ├── /kiwoom/indicators  RSI, BB, 거래량, MA 계산
    ├── /kiwoom/quote       호가비 계산
    └── DART 공시 악재 필터
    ↓
SELL: 신호 3개 이상 → trade.py subprocess 실행
BUY:  신호 N개 이상 → trade.py subprocess 실행
    ↓
/signal/receive HMAC 검증 → 키움 지정가 주문 접수
    ↓
DISCORD_MSG → Discord DM 자동 전송
```
