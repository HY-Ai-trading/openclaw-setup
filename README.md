# openclaw-setup

키움증권 자동매매 봇. 기술 지표 분석 후 자동 매수/매도 주문, Discord DM으로 결과 전송.

> **`python-only/` — 현재 개발자가 사용 중인 프로덕션 버전**

| 폴더 | 상태 | AI | 뉴스 | 비용 |
|------|------|-----|------|------|
| `python-only/` | ✅ **현재 사용 중** | 없음 | DART 공시만 | 없음 |

| `ai-news/` | 📚 레퍼런스 | OpenClaw | 매회 검색 | 더 높음 |

## python-only — 현재 프로덕션

### 내장 기능

- **기술 지표 분석**: RSI, 볼린저밴드, MA 골든크로스, 거래량비, 호가비(매수/매도잔량)
- **시장 상태 판단**: KODEX200 RSI·등락률으로 과열 여부 확인
- **DART 공시 필터**: 악재 공시 종목 매수 제외, 호재 공시 신호 +1
- **2단계 지표 조회**: 호가비 상위 5개 + 보유종목만 indicators 조회 (API 절약)
- **장종료 감지**: 5회 거부 누적 시 당일 스캔 자동 중단, 다음 거래일 자동 재개
- **공휴일 필터**: 법정공휴일 + 근로자의 날(KRX 휴장) 자동 SKIP
- **Discord 알림**: 매수/매도 체결, HOLD(최고 신호 포함) 2분마다 전송
- **주문 재시도**: 네트워크 오류 시 3회 재시도

### 매수 조건 (7개 중 3개↑ 충족)

| 조건 | 기준 |
|------|------|
| RSI | ≤ 45 |
| 거래량 | ≥ 1.3x |
| 당일 하락 | ≤ -1.0% |
| BB 하단 | 하단 터치 |
| MA 골든크로스 | MA5 > MA20 |
| 호가비 | ≥ 1.2 |
| buy_rt | ≥ 130 |

- 매도 신호 이미 있는 종목은 매수 안 함
- DART 악재 공시 종목 제외

### 매도 조건

| 조건 | 기준 | 필요 신호 |
|------|------|----------|
| 수익 실현 | ≥ +7% | 1개 단독 |
| RSI 과매수 | ≥ 80 | 2개↑ |
| 급등 | 당일 ≥ +7% | 2개↑ |
| BB 상단 | 상단 터치 | 2개↑ |
| 매도잔량 우위 | 호가비 < 0.5 | 2개↑ |
| 비상손절 | ≤ -30% | 즉시 |

### 실행 방법

```bash
cd python-only
cp ../.env.example .env
nano .env          # 환경변수 입력
pip install httpx python-dotenv holidays
bash cron-setup.sh # cron 등록
```

등록되는 cron 스케줄:
```
50 8   * * 1-5   scan.py 직접 실행        # 장 시작 전 8:50
*/2 9-15 * * 1-5  check_time.py → scan.py  # 장중 2분마다
```

수동 실행:
```bash
python3 scan.py
```

## ai-news — 레퍼런스 (미사용)

순수 AI 방식. scan.py가 데이터만 출력하고 판단은 전부 OpenClaw(gpt-4.1-mini)가 담당.
매 실행마다 구글 뉴스 RSS 검색 → 뉴스 + 기술지표 종합 판단 → 자동 주문.

> 현재 개발자는 사용하지 않음. 구조 참고용.

실행 방법: `ai-news/HOW_IT_WORKS.md` 참조

## 공통 환경변수 (.env)

| 키 | 설명 |
|----|------|
| `TRADING_SERVER_URL` | 키움 API 브릿지 서버 주소 |
| `SIGNAL_SECRET_KEY` | API 서버 인증 키 |
| `DISCORD_BOT_TOKEN` | Discord Bot 토큰 |
| `DISCORD_USER_ID` | DM 수신자 Discord 사용자 ID |
| `DART_API_KEY` | [OpenDART](https://opendart.fss.or.kr) 발급 |
